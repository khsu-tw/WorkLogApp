## 🚀 快速開始

### 步驟 1: 下載 PocketBase
訪問 https://pocketbase.io/docs/ 下載 Windows 版本

### 步驟 2: 啟動 PocketBase
```bash
pocketbase.exe serve
# 開啟 http://127.0.0.1:8090/_/
```

### 步驟 3: 設定 Collection
參考 [POCKETBASE_SETUP.md](POCKETBASE_SETUP.md)

### 步驟 4: 遷移數據
```bash
python migrate_data.py
```

### 步驟 5: 啟動應用
```bash
python app.py
```

## ⚠️ 已知問題和解決方案

### 問題: 遷移腳本返回 400 錯誤

**原因**: PocketBase API Rules 或欄位驗證

**解決方案**:
1. 在 PocketBase Admin UI 設定 API Rules 為公開
2. 確認所有必需欄位已創建
3. 使用管理員認證進行遷移

詳見 [POCKETBASE_SETUP.md](POCKETBASE_SETUP.md)
