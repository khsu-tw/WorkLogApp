## 🚀 部署說明

### Windows 執行檔部署
1. 壓縮整個 `dist\WorkLog` 資料夾
2. 分發給使用者
3. 解壓縮後執行 WorkLog.exe
4. 自動彈出 PocketBase 設定對話框

### PocketBase 部署
1. 下載 PocketBase: https://pocketbase.io/docs/
2. 啟動: `pocketbase.exe serve`
3. 訪問管理界面: http://127.0.0.1:8090/_/
4. 創建 worklog collection (參考 POCKETBASE_SETUP.md)
5. 運行遷移腳本: `python migrate_data.py`

### 源碼部署
```bash
# 1. 解壓縮 WorkLogApp_v0.9.8_final.zip
# 2. 安裝依賴
pip install -r requirements.txt

# 3. 啟動應用
python app.py

# 訪問 http://localhost:5000
```

## 🎯 使用方式

### 方式 1: 本地 PocketBase (推薦)
```
1. 啟動 PocketBase (http://127.0.0.1:8090)
2. 啟動 WorkLog.exe 或 python app.py
3. 配置 PocketBase URL
4. 開始使用
```

### 方式 2: 僅本地 SQLite
```
1. 啟動 WorkLog.exe
2. 設定對話框選擇 "Use Local Storage Only"
3. 開始使用 (無需 PocketBase)
```
